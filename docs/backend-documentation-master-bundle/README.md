## How to use

1. Start with `00-MASTER_BACKEND_DOCUMENTATION_INDEX.md`.
2. Use `01-ROLE_BASED_API_MAP.md` to understand routes by role.
3. Use `02-PERMISSION_SECURITY_MATRIX.md` for cross-role permission checks.
4. Use `03-POSTMAN_SQL_FILES_INDEX.md` to find the right Postman and SQL files.
5. Use `04-PRODUCTION_READINESS_CHECKLIST.md` before live deployment.
6. Use `05-KNOWN_REGRESSIONS_AND_REVIEW_POINTS.md` to review issues discovered during documentation.
7. Use the ZIP files inside `all-batch-zips/` as the source packages for each batch.
